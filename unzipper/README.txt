Il codice in PHP consente di trovare ed estrarre TUTTI i file ZIP all'interno di una directory, anche quelli contenuti a loro volta in un archivio ZIP.


La cartella "contiene" ed i file in formato ZIP servono per testare il funzionamento del codice.

Archivio1.zip contiene al suo interno un'ulteriore file in formato ZIP che verrà estratto separatamente.
La cartella "contiene" ha al suo interno diverse cartelle che contengono file e archivi ZIP.


Verrà creata una nuova cartella "filestratti" che ospiterà tutti i file estratti.
